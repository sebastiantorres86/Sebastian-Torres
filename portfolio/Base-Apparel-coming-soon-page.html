Base Apparel coming soon page

My challenge was to build out this coming soon page and get it looking as close to the design as possible. The users should be able to view the optimal layout for the site depending on their device's screen size, see hover states for all interactive elements on the page, receive an error message when the form is submitted if: The input field is empty, the email address is not formatted correctly. I used HTML5, along with CSS Flexbox and Vanilla JavaScript.
